HieknInfoboxService Settings
Properties
  • baseUrl: string
    获取infobox时的sdk服务地址前缀
  • data: {}
    (Optional) 获取infobox时的默认发送参数
  • enableLink: boolean
    是否允许infobox内的链接，默认值为 false
  • href(id, instance): Function
    (Optional) 当enableLink为True时有效, 点击超链接时触发
  • kgName: string
    获取infobox时的kgName
  • selector: string
    指定生成infobox的Dom选择器

Properties
